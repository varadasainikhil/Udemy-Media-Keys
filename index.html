Udemy Media Keys

A lightweight Chrome extension that maps your keyboard media keys to Udemy course player controls.

What it does

  • Play / pause with the play/pause media key
  • Rewind via previous-track media key
  • Skip forward via next-track media key
  • Works on Udemy course video pages (https://*.udemy.com/*)

How to use

  1. Install the extension from Chrome Web Store.
  2. Open any Udemy video lesson page.
  3. Press media keys on your keyboard; controls are handled automatically.

Privacy

No data is collected, stored, or transmitted by this extension.

All operation is local within the browser on Udemy pages only.
