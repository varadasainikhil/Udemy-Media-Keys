Udemy Media Keys

Privacy Policy

This extension is designed solely to map keyboard media key events (play/pause/next/previous) to the Udemy video player controls on Udemy pages.

It runs entirely in the user's browser and does not collect, store, or transmit any personal data, analytics, or tracking information to any external service.

No remote code is loaded at runtime; all JavaScript is included in the extension package.

Host permission is only for https://*.udemy.com/* to enable the extension to work on Udemy course video pages.

Data Usage

  • Personally identifiable information: none
  • Health information: none
  • Financial/payment information: none
  • Authentication information: none
  • Personal communications: none
  • Location: none
  • Web history: none
  • User activity: none
  • Website content: none collected

Because this extension does not collect user data, there is no data shared with third parties.
